# 📚 FlashCards 單字學習工具

> 互動式英中單字卡學習系統，支援字卡練習、測驗模式與熟練度追蹤

🔗 **線上體驗：** https://winnieshih1107.github.io/Vocabulary/

---

## 工作報告

### 專案概述

本工具為一款純前端單字學習應用，以 HTML / CSS / Vanilla JavaScript 開發，無需安裝任何套件或後端伺服器，可直接在瀏覽器中運行。所有學習紀錄透過 `localStorage` 儲存於本機，資料不會上傳至任何伺服器。

### 功能特色

| 功能 | 說明 |
|------|------|
| 📥 單字匯入 | 每行一個，格式 `英文, 中文`，支援批次貼上 |
| 📤 單字匯出 | 匯出為 `.txt` 檔案，方便備份或轉移 |
| 📚 字卡練習 | 3D 翻轉動畫字卡，支援英文／中文正面切換 |
| 🔊 語音發音 | 英文（Brian）與中文（Zhiwei）TTS 發音 |
| ✏️ 測驗模式 | 四選一選擇題，支援「看英文選中文」及「看中文選英文」 |
| 📊 熟練度追蹤 | 依學習次數與答對率自動標記：初學 / 練習中 / 熟練 |
| ✓ 熟練單字管理 | 可單獨或批次移除已熟練的單字，聚焦待加強項目 |
| ⌨️ 鍵盤操作 | 字卡模式支援 ← → 切換、空白鍵翻面 |
| 📱 響應式設計 | 支援桌機與行動裝置 |

### 技術實作

- **前端框架：** 無框架，純 Vanilla JavaScript
- **樣式：** CSS Custom Properties（設計 Token 系統）
- **字型：** Google Fonts — Nunito + Noto Sans TC
- **語音：** StreamElements TTS API（主）/ Web Speech API（備援）
- **儲存：** `localStorage`（key: `flashcards_words`）
- **熟練判定：** 學習 ≥ 5 次 且 答對率 ≥ 80%

### 開發紀錄

- 建置互動式字卡翻轉動畫（CSS 3D transform + perspective）
- 實作學習進度追蹤系統（seen / correct 計數器）
- 整合 TTS 語音功能（API 失敗時自動切換 Web Speech 備援）
- 設計熟練度管理面板，支援移除已熟練單字
- 鍵盤快捷鍵支援（提升使用效率）

## 使用說明

### 第一步：匯入單字

1. 點選頁面上方「**📥 匯入單字**」分頁
2. 在文字框中輸入單字，**每行一個**，格式為：

   ```
   apple, 蘋果
   beautiful, 美麗的
   computer, 電腦
   ```

3. 點擊「**✓ 匯入單字**」按鈕完成匯入

> 💡 重複的單字會自動略過，不會重複新增

---

### 第二步：字卡練習

1. 點選「**📚 字卡練習**」分頁
2. **點擊字卡**翻面查看答案
3. 使用左右箭頭切換上一張 / 下一張
4. 點「**🎲 隨機**」打亂順序；點「**↩ 重置**」回到原始順序
5. 點「**英文正面**」或「**中文正面**」切換練習方向
6. 點「**🔊 英文發音**」或「**🔊 中文發音**」聆聽發音

**鍵盤快捷鍵**

| 按鍵 | 功能 |
|------|------|
| `←` | 上一張 |
| `→` | 下一張 |
| `空白鍵` / `Enter` | 翻面 |

---

### 第三步：測驗模式

1. 點選「**✏️ 測驗模式**」分頁
2. 選擇題型：
   - **看英文 → 選中文**
   - **看中文 → 選英文**
3. 每次從題庫隨機抽取最多 10 題
4. 點擊選項後立即顯示對錯，綠色為正確答案
5. 完成後顯示成績與正確率

---

### 熟練度系統

每次翻字卡或回答測驗題，系統會自動記錄學習次數：

| 標記 | 條件 |
|------|------|
| 未學習 | 尚未練習過 |
| 🟡 初學 | 練習 1–2 次 |
| 🔵 練習中 | 練習 3–4 次 |
| 🟢 ✓ 熟練 | 練習 ≥ 5 次 且 答對率 ≥ 80% |

當有單字達到「熟練」，匯入頁面會出現「**✓ 查看並移除已練熟單字**」按鈕，可將熟練的單字從清單中移除，集中練習尚未熟悉的單字。

---

### 匯出與備份

- 點「**📤 匯出單字**」可將目前所有單字存成 `.txt` 檔案
- 格式與匯入相同，可隨時重新匯入
